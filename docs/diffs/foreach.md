# Where is foreach?

In the **hapet** programming language, there is no ***foreach*** keyword. Simply use the ***for*** loop:  
```csharp
string str = "Hello World!";
for (char c in str)
{
    Console.Write(c);
}
```

Why is it this way? Well, just because :)
