# hapet check  

## Использование

Команда ```hapet check``` может быть использована для проверки работоспособности компилятора **hapet**. Пример использования:
```bash title="bash"
hapet check
```  

Пример вывода ```hapet check```:
```bash title="bash"
user>hapet check
Linker: OK
Win-x64 libraries: OK
Message files: OK

Check done in 0,26 seconds
Result: hapet is OK
```
