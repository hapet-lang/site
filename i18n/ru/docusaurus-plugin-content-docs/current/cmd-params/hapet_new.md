# hapet new  

## Использование

Команда ```hapet new``` может быть использована для создания проектов **hapet**. Пример использования:
```bash title="bash"
hapet new console -n ConsoleTest
```  

Для просмотра доступных шаблонов и параметров запустите команду:
```bash title="bash"
hapet new -h
```   

## Доступные шаблоны проектов  

- ```console``` - шаблон проекта для консольного приложения.
- ```classlib``` - шаблон проекта для библиотеки классов.  

## Аргументы  

- ```-n|--name <PROJECT_NAME>``` - название создаваемого проекта. Если имя не задано - будет использовано стандартное имя проекта заданного шаблона.
