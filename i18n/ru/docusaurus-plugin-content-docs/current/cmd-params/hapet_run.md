# hapet run  

## Использование

Команда ```hapet run``` может быть использована для запуска проекта. Пример использования:
```bash title="bash"
hapet run /project/MyTestProject.hptproj
```  

Если имя запускаемого проекта не указано, **hapet** попытается найти проект в текущей открытой директории.  

Для просмотра доступных параметров запустите команду:
```bash title="bash"
hapet run -h
```  

## Аргументы  

- ```-t|--target <PLATFORM>``` - платформа, под которую будет собран проект. Если целевая платформа не указана - проект будет собран под текущую платформу.  
  Список доступных платформ: ```win-x64```, ```win-x86```, ```linux-x64```, ```linux-x86```.  
- ```--verbose``` - детальный вывод при сборке проекта.  
- ```--debug|--release``` - конфигурация сборки проекта. Если конфигурация не указана, то используется ```--debug```.
