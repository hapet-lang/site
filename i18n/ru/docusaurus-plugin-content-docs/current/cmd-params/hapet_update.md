# hapet update  

## Использование

Команда ```hapet update``` может быть использована для обновления компилятора **hapet** до последней версии. Пример использования:
```bash title="bash"
hapet update
```  

Для просмотра доступных параметров запустите команду:
```bash title="bash"
hapet update -h
```   

:::warning
Обратите внимание - ```hapet updater``` запускает после себя процесс для замены скачанных файлов компилятора.  
Этот процесс может занимать некоторое количество времени (обычно несколько секунд).   
Во время замены файлов желательно не пользоваться компилятором.   
Для большего контроля обновления компилятора смотрите параметр ```--manual```. 
:::

## Аргументы  

- ```--manual``` - с этим параметром команда ```hapet update``` не будет запускать после себя программу для замены файлов компиятора.   
  Поэтому, для успешного обновления компилятора, пользователю будет необходимо вызвать команду ```hapet-replacer```.
