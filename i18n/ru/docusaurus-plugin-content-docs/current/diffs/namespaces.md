# Пространства имен

В языке программирования **hapet** пространства имен могут задаваться только в одну линию:  
```csharp
using ...;

namespace CustomNamespace;

...
```

Заданное пространство имен будет применено ко всему файлу. На данный момент возможно наличие только одного указания пространства имен.  

Указание пространства имен с помощью фигурных скобок ```namespace { ... }``` **недопустимо**.  

Если пространство имен не было указано, оно будет автоматически обозначено как:  
```csharp
namespace НазваниеПроекта.Путь.До.Файла;
```
