# Ключевые слова  

Язык **hapet** предоставляет новые ключевые слова.  

## unreflected

Ключевое слово ***unreflected*** используется как модификатор доступа наподобие ***public***/***protected***/***internal***/***private***. Внутри одной сборки модификатор доступа ***unreflected*** имеет те же свойства, что и ***private***. Отличаются они тем, что декларация с ключевым словом ***unreflected*** не видна вне текущей сборки, к ней нельзя обратиться даже с помощью рефлексии. 

```csharp
unreflected void SuperPrivateMethod()
{
    ...
}
```

## inline

Ключевое слово ***inline*** может быть использовано для функций, которые должны быть раскрыты в контексте вызова. 

```csharp
public static inline int Min(int a, int b)
{
    return (a <= b) ? a : b;
}
```
