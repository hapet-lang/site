# Статичные и константные поля

В языке программирования **hapet** поля с модификаторами ***static*** и ***const*** будут вести себя не так, как в **C#**. При обращении к таким полям в **hapet** будет вызываться один из методов *get* или *set*.  

Данное решение накладывает ограничение на использование статичных полей как аргумент для ссылочного паратера. То есть следующий код не будет работать:
```cs
public class TestClass
{
    public static int TestField = 0;

    public void MethodCaller()
    {
        MethodCallee(ref TestField);
    }

    private void MethodCallee(ref int v)
    {
        v = 5;
    }
}
```

Это сделано для поддержания совместимости работы динамических библиотек. В дальнейшем планируется добавление модификаторов ***reff*** и ***outt*** для поддержки использования статических полей и свойств для ссылочных параметров.  

Данное ограничение также накладывается на использование модификатора ***out***.
