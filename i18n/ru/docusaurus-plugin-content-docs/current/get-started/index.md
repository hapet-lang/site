# Начало работы с hapet

## Установка

Перед началом работы с **hapet** необходимо скачать [собранный установщик](https://hapetlang.com/#downloads) компилятора или [собрать компилятор самому](https://github.com/hapet-lang/hapet?tab=readme-ov-file#running-from-source) (нафига?).  

После установки можно проверить, что **hapet** работоспособен. Для этого в терминале запустите команду:
```bash title="bash"
hapet check
```  

[Подробнее про использование **hapet check**](../cmd-params/hapet_check).

## Создание проекта  

Для создания проекта вы можете воспользоваться командой ```hapet new```. Пример создания проекта консольного приложения:  
```bash title="bash"
hapet new console -n MyProject
```  

После выполнения команды будет создана папка *MyProject* с файлами проекта.   
[Подробнее про использование **hapet new**](../cmd-params/hapet_new).

## Запуск проекта  

Для запуска проекта перейдите в папку с проектом (в папку, где лежит файл с расширением *.hptproj*) и запустите команду:  
```bash title="bash"
hapet run
```  

[Подробнее про использование **hapet run**](../cmd-params/hapet_run).

## Работа с проектом  

Для удобной (нет) работы с проектом было создано [расширение для редактора кода *vs-code*](https://marketplace.visualstudio.com/items?itemName=crackanddie.hapet-vscode).  

После установки расширения, можно открыть папку с проектом, где лежит файл с расширением *.hptproj*.  

:::warning
Обратите внимание - расширение **hapet** в *vs-code* работает только, если открыта папка с файлом проекта (файл с расширением *.hptproj*)! 
:::
