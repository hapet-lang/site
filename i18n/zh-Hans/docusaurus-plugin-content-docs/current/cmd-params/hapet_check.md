# hapet check  

## 用法

```hapet check``` 命令可用于验证 **hapet** 编译器的功能。使用示例：
```bash title="bash"
hapet check
```  

```hapet check``` 的输出示例：
```bash title="bash"
user>hapet check
Linker: OK
Win-x64 libraries: OK
Message files: OK

Check done in 0,26 seconds
Result: hapet is OK
```
